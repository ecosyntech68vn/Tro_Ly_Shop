# 15 PROMPT MẪU — DÙNG NGAY VỚI COPYWRITER VIỆT PRO

Đây là những câu lệnh (prompt) bạn có thể copy-paste vào Claude/ChatGPT ngay sau khi cài đặt xong. Không cần sửa gì — chạy là có kết quả.

---

## THỜI TRANG

**Prompt 1 — Bài Facebook bán áo thun nam:**
```
Tôi bán áo thun nam cotton, giá 199k, chất liệu mềm mịn thoáng mát.
Khách hàng: nam 25-35 tuổi, dân văn phòng.
Giúp tôi viết 1 bài Facebook dạng cảm xúc, kết thúc bằng CTA inbox.
```

**Prompt 2 — TikTok quảng cáo váy nữ:**
```
Viết kịch bản TikTok 20 giây cho shop thời trang nữ.
Sản phẩm: đầm suông công sở, giá 399k.
Hook mạnh 2 giây đầu, nhấn dáng người và chất vải.
```

## F&B

```
Giọng ấm áp, kể câu chuyện vì sao mở quán.
Có ưu đãi: giảm 20% tuần đầu.
```

**Prompt 4 — Mô tả Shopee cho set quà Tết:**
```
Viết mô tả Shopee cho set quà Tết 2027: bánh mứt kẹo + trà, giá 299k.
Có hình hộp đẹp, miễn phí giao thiệp tay.

```

## MẸO: TỰ TẠO PROMPT

```
Kênh: [Facebook / TikTok / Shopee].
Giúp tôi viết [dạng bài].
Yêu cầu thêm: [giọng văn, điểm nhấn, khuyến mãi].
```
